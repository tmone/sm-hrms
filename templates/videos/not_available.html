Video Management Not Available

The video management feature is currently not available. This feature requires additional video processing libraries and models to be installed.

To enable video management:

  • 1. Install video processing libraries (OpenCV, FFmpeg, etc.)
  • 2. Configure video storage and processing settings
  • 3. Set up background processing with Celery (optional)
  • 4. Restart the application
